print("Hello, World!")

⚠️

들여쓰기가 잘못되면 IndentationError가 발생합니다. 일반적으로 스페이스 4칸 또는 탭 1개를 사용하세요. 같은 블록 안에서는 반드시 동일한 들여쓰기를 써야 합니다.
🔢 변수 & 연산자
파이썬은 타입을 직접 쓰지 않아도 됩니다
📝 변수 선언
☕ 자바 — 타입 명시 필수
int num = 123; double num2 = 123.4; String str = "hello"; boolean isTrue = false;
🐍 파이썬 — 타입 자동 추론
num = 123 num2 = 123.4 str = "hello" isTrue = False # 대문자 F!
➗ 나누기 연산자 비교
//
몫 (정수 나눗셈)
5 // 3 → 1
%
나머지
5 % 3 → 2
/
나누기 (실수)
5 / 3 → 1.666
divmod
몫 + 나머지 동시에
divmod(5,3) → (1,2)
구분 ☕ 자바 🐍 파이썬
정수 나누기 int x = 5 / 3; → 1 5 // 3 → 1
나머지 5 % 3 → 2 5 % 3 → 2
실수 나누기 5.0 / 3 → 1.666 5 / 3 → 1.666
입력 받기 Scanner.nextInt() int(input("입력: "))
💡
파이썬 핵심 포인트: True / False는 반드시 대문자로 시작! 자바의 true / false와 다릅니다. 파이썬에서 / 연산은 항상 실수를 반환합니다.
🔄 제어 흐름
if / for / while — 파이썬은 훨씬 읽기 쉽습니다
if
조건문 (if / elif / else)
☕ 자바
if (a > b) { println("a가 크다"); } else if (a < b) { println("a가 작다"); } else { println("같다"); }
🐍 파이썬
if a > b: print("a가 크다") elif a < b: print("a가 작다") else: print("같다")
🔑 else if → elif 로 변경!
for
반복문 (for)
☕ 자바
for (int i=0; i<5; i++) { System.out.println(i); } // Enhanced for for (타입 변수 : 컬렉션) { ... }
🐍 파이썬
for i in range(5): print(i) # 컬렉션 순회 for 변수 in 컬렉션: ...
🔑 range(5) = 0, 1, 2, 3, 4 순서대로
while
반복문 (while)
☕ 자바
int i = 0; while (i < 10) { System.out.printf( "%d번째\n", i); i++; }
🐍 파이썬
i = 0 while i < 10: print(f"{}번째".format(i)) i += 1
🔑 i++ 없음 → i += 1 사용
파이썬 제어문 핵심
: 조건/반복 끝에 콜론 필수
elif 자바의 else if → 파이썬은 elif
in for 반복은 in 키워드 사용
{ } 중괄호 사용 안 함!
📦 자료구조
파이썬의 4가지 핵심 자료구조
[ ]
list 리스트
순서 있고 수정 가능한 목록
arr = [1, 3, 5, 7, 9] arr.append(11) # 추가 arr.extend([13]) # 여러 개 arr[0] = 2 # 수정 del arr[0] # 삭제 print(arr[0]) # 조회 print(len(arr)) # 길이
( )
tuple 튜플
수정 불가능한 리스트 (읽기 전용)
t = (1, 2, 3) print(t[0]) # 조회만 가능 # t[0] = 10 → 오류! # divmod도 튜플 반환 q, r = divmod(5, 3)
{ }
set 집합
중복 자동 제거 + 집합 연산
s = {"apple","banana","apple"} # → {"apple","banana"} 중복제거! s.add("kiwi") # 추가 s.update({"berry"}) # 여러 개 print("apple" in s) # 포함 여부
{:}
dict 딕셔너리
key-value 쌍 (자바의 Map)
d = {"apple": 3, "banana": 5} d["kiwi"] = 7 # 추가/수정 print(d["apple"]) # 조회 → 3 print(d.keys()) # 키 목록
자료구조 기호 순서 중복 수정 특징
list [ ] ✅ 있음 ✅ 허용 ✅ 가능 가장 많이 사용
tuple ( ) ✅ 있음 ✅ 허용 ❌ 불가 읽기 전용
set { } ❌ 없음 ❌ 제거 ✅ 가능 중복 자동 제거
dict {k:v} ✅ 있음 키 불가 ✅ 가능 key로 조회
⚙️ 함수 & 클래스
파이썬은 def 하나로 함수 끝! 클래스도 훨씬 간결합니다
📌 함수 (Function / Method)
☕ 자바 — 반환타입 명시 필수
int f(int a, int b) { return a + b; }
🐍 파이썬 — def 키워드만!
def f(a, b): return a + b
🏗️ 클래스 (Class)
☕ 자바
class Good { List<Integer> li = List.of(1,2); } public class ClassTest { public static void main(...) { Good good = new Good(); System.out.print( good.li.get(0)); } }
🐍 파이썬
class Good: # 클래스 li = [1, 2] # 필드 g = Good() # 인스턴스 생성 print(g.li[0]) # 조회
✂️ 슬라이싱 (Slicing) — 파이썬만의 강력한 기능
리스트 슬라이싱
🐍 파이썬
a = [1, 2, 3, 4, 5] b = a[2:4] # [3, 4] c = a[-1] # 5 (마지막) d = a[::-1] # 뒤집기
문자열 슬라이싱
🐍 파이썬
a = "good" print(a[0:2]) # "go" print(a[-2:]) # "od"
슬라이싱 규칙 a[start:end]
a[2:4]
인덱스 2~3 추출
→
0
1 1
2 2
3 3
4 4
5
← 값
← 인덱스
✅ end 인덱스는 포함되지 않습니다. a[2:4] = 인덱스 2, 3번 요소만 추출
🧮 집합 연산 & 컴프리헨션
파이썬만의 강력한 기능들
⭕ set 집합 연산
a & b
교집합
두 집합 모두에 있는 것
a - b
차집합
a에만 있고 b엔 없는 것
a | b
합집합
a 또는 b에 있는 것 전부
a ^ b
대칭차집합
겹치지 않는 것만 남김
예시: a = {1,2,3,4} / b = {3,4,5,6}
a & b
{3, 4}
a - b
{1, 2}
a | b
{1,2,3,4,5,6}
a ^ b
{1,2,5,6}
⚡ 컴프리헨션 (Comprehension)
☕ 자바 — 복잡한 Stream API
List<Integer> squares = IntStream.range(0, 10) .map(x -> x * x) .boxed() .collect( Collectors.toList());
🐍 파이썬 — 한 줄로 끝!
squares = list(map( lambda x: x**2, range(10) )) # 더 파이썬스럽게: squares = [x**2 for x in range(10)]
