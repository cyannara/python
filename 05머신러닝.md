# 머신러닝

- 사람이 규칙을 직접 코딩하지 않고 `데이터를 통해 컴퓨터가 규칙(패턴)을 학습하게 하는 것`
- 모델을 만드는 과정

`모델`

- 모델은 데이터를 보고 학습한 패턴(`규칙덩어리`)
- 모델 내부에는 가중치(weight), 규칙, 패턴 정보가 저장 ex) 선형회귀: y = 3x + 5

```
데이터 수집
  ↓
전처리
  ↓
모델(알고리즘) 선택
  ↓
머신러닝 학습
  ↓
모델 생성
  ↓
예측 수행
```

`실제 코드흐름`

```
 ↓
입력(x) + 정답(y)       ->       모델학습(Training)        ->      예측(Inference)
                             model.fit(x_train,y_train)         model.predict(x_test)


수많은 이메일 데이터    ->       모델이 패턴 학습          ->     새 이메일 자동 판별

```

`모델학습단계(Training)`

```
입력데이터 -> 예측수행 -> 정답과 비교 -> 오차계산 -> 가중치 수정 -> 반복
```

## 지도학습

- 지도학습은 크게 2가지로 분류(Classification)와 회귀(Regression)로 나뉨. 분류는 정답이 카테고리인 문제이고 회귀는 정답이 숫자인 문제

``지도학습 대표 알고리즘`

| 분류 | 알고리즘(대표 모델)                | 특징                       | 예시                                      |
| :--- | :--------------------------------- | :------------------------- | :---------------------------------------- |
| 분류 | Logistic Regression, Decision Tree | 결과가 범주형(YES/NO)      | 이메일 스팸 판별,고객 이탈 예측,상품 추천 |
|      | Random Forest,Neural Network       |                            |                                           |
| 회귀 | Linear Regression,Ridge/Lasso      | 연속적인 숫자 예측(실수값) | 집값 예측,광고비와 매출                   |
|      | XGBoost Regression                 |                            |                                           |

## 비지도학습

- 정답(label) 없이 데이터 패턴만 찾는 방식
- 예시: 고객 구매 데이터 -> 비슷한 고객끼리 그룹화
- 대표작업: 군집화(Clustering)-비슷한 데이터 묶기, 자원축소-데이터 압축, 이상탐지-이상한 패턴 찾기
- 대표알고리즘: K-Means, PCA, DBSCAN

## scikit-learn

scikit-learn의 알고리즘 치트 시트로 불린다. scikit-learn을 이용하여 기계학습을 실행할 때, 자신이 하고 싶은 분석(분류/회귀/클러스터링 등)에 대해서 적당한 모델을 선택

```
1) 데이터 세트 읽어 들이기

2) 훈련 데이터와 테스트 데이터 준비

3) 학습

4) 모델의 평가
```
