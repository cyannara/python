## 인공지능

- 사람처럼 판단, 예측, 추천, 생성하는 기술 전체
- 챗봇, 음성인식, 자율주행, 이미지 생성, 추천 시스템

`전체 구조`

```
인공지능(Artificial Intelligence)                  사람처럼 판단/예측.추천하려는 전체 분야
 └── 머신러닝(Machine Learning)                    데이터를 통해 규칙(패턴)을 학습하는 방법
      ├── 지도학습(Supervised Learning)            정답(Label)이 있는 데이터를 학습하는 방식
      │     ├── 분류(Classification)               정답이 카테고리인 문제
      │     └── 회귀(Regression)                   정답이 숫자인 문제
      │
      ├── 비지도학습(Unsupervised Learning)        정답(label) 없이 데이터 패턴만 찾는 방식
      │
      └── 딥러닝(Deep Learning)                    머신러닝 중에서 신경망을 깊게 쌓은 방식
            ├── CNN(Convolutional Neural Network)  이미지 처리에 특화된 딥러닝 모델
            ├── RNN(Recurrent Neural Network)      순서(sequence) 데이터 처리 모델. 자연어/시계열
            └── Transformer                        현재 AI의 핵심 구조. 문장 안에서 중요한 단어끼리 관계를 학습
```

`서비스 예시`

- 유투브 추천 : 머신러닝 + 딥러닝
- ChatGPT : Transformer 기반 LLM
- 자율주행 : CNN + Transformer
- 주가예측 : RNN/LSTM + Transformer

`방식`

```
학습(Training)   -> 모델 생성  -> 추론(Inference)  -> 실제 예측 서비스
```

### 실무에서 어떻게 이어지는가

예: 쇼핑몰 추천 시스템

```
데이터 수집 -> 머신러닝/딥러닝 모델 학습 -> 사용자 행동 패턴 학습 -> 추론 단계에서 추천 수행
```

예: 집값 예측

```
회귀 모델 학습 -> 새 집 정보 입력 -> 가격 추론
```

예: 스팸 메일

```
분류 모델 학습 -> 메일 입력 -> 스팸 여부 추론
```

### 머신러닝

- 사람이 규칙을 직접 코딩하지 않고 `데이터를 통해 컴퓨터가 규칙(패턴)을 학습하게 하는 것`

### 지도학습

- 지도학습은 크게 2가지로 분류(Classification)와 회귀(Regression)로 나뉨. 분류는 정답이 카테고리인 문제이고 회귀는 정답이 숫자인 문제

`핵심흐름`

```
입력(x) + 정답(y)       ->       모델학습(Training)        ->      예측(Inference)
                             model.fit(x_train,y_train)         model.predict(x_test)


수많은 이메일 데이터    ->       모델이 패턴 학습          ->     새 이메일 자동 판별
```

`모델학습단계(Training)`

```
입력데이터 -> 예측수행 -> 정답과 비교 -> 오차계산 -> 가중치 수정 -> 반복
```

#### 지도학습 대표 알고리즘

| 분류 | 알고리즘(대표 모델)                | 특징                       | 예시                                      |
| :--- | :--------------------------------- | :------------------------- | :---------------------------------------- |
| 분류 | Logistic Regression, Decision Tree | 결과가 범주형(YES/NO)      | 이메일 스팸 판별,고객 이탈 예측,상품 추천 |
|      | Random Forest,Neural Network       |                            |                                           |
| 회귀 | Linear Regression,Ridge/Lasso      | 연속적인 숫자 예측(실수값) | 집값 예측,광고비와 매출                   |
|      | XGBoost Regression                 |                            |                                           |

### 비지도학습

- 정답(label) 없이 데이터 패턴만 찾는 방식
- 예시: 고객 구매 데이터 -> 비슷한 고객끼리 그룹화
- 대표작업: 군집화(Clustering)-비슷한 데이터 묶기, 자원축소-데이터 압축, 이상탐지-이상한 패턴 찾기
- 대표알고리즘: K-Means, PCA, DBSCAN

### 딥러닝

딥러닝은 사람 뇌의 뉴런 구조를 흉내낸 인공신경망(ANN)을 사용합니다.

- 데이터가 많을수록 강력
- 이미지/음성/NLP에서 뛰어남
- GPU 사용이 많음

### 머신러닝과 딥러닝 차이

| 항목      | 머신러닝       | 딥러닝           |
| --------- | -------------- | ---------------- |
| 특징 추출 | 사람이 직접    | 모델이 자동      |
| 데이터 양 | 적어도 가능    | 많이 필요        |
| 성능      | 일반 문제 강함 | 복잡한 문제 강함 |
| 속도      | 빠름           | 느림             |
| 하드웨어  | CPU 가능       | GPU 권장         |
| 대표 분야 | 표 데이터      | 이미지/음성/NLP  |

#### 딥러닝 예시

| 분야      | 예시                 |
| :-------- | :------------------- |
| 이미지    | 고양이 판별          |
| 음성      | 음성 인식            |
| 자연어    | 챗봇, 번역           |
| 생성형 AI | ChatGPT, 이미지 생성 |

#### 딥러닝 대표 프레임워크

| 프레임워크 | 특징     |
| :--------- | :------- |
| TensorFlow | 구글     |
| PyTorch    | 메타     |
| Keras      | 쉬운 API |

#### 딥러닝 구조

은닉층이 깊어질수록 "Deep Learning"

```
입력층
  ↓
은닉층(hidden layer)
  ↓
출력층
```

### CNN (Convolutional Neural Network)

이미지 처리에 특화된 딥러닝 모델

동작

```
이미지
 ↓
필터(Convolution)
 ↓
특징 추출
 ↓
분류
```

### RNN (Recurrent Neural Network)

- 순서(sequence) 데이터 처리 모델
- 이전 정보를 기억하며 처리
- 예시 : 문장 번역, 음성 인식, 시계열-주가예측
- 문제점 : 문장이 길어질수록 기억 약화 --> 이를 개선한 것이 LSTM, GRU

### Transformer

- 현재 AI의 핵심 구조
- 문장 안에서 중요한 단어끼리 관계를 학습
- ChatGPT도 Transformer 기반입니다.
- 활용분야 : 챗봇-ChatGPT, 번역-DeepL, 이미지 생성-Vision Transformer, 음성-Whisper

시대흐름

```
전통 머신러닝
   ↓
딥러닝 등장
   ↓
CNN (이미지 혁명)
   ↓
RNN (자연어/시계열)
   ↓
Transformer (현재 AI 핵심)
```
