# 구글 코랩(colaboratory)

- Collaborative(협업하는) + Laboratory(실험실) => colaboratory => Colab
- 구글 코랩은 Google에서 제공하는 Python Jupyter Notebook 앱이다.
- 데이터 분석에 사용되는 Tensorflow, keras, Matplotlib, Scikit-learn, Pandas 등의 패키지가 기본적으로 설치되어 있음
- GPU를 무료로 사용: 런타임 유형변경 -> 하드웨어 가속기를 GPU로 설정

## 단축키

| 단축키    | 설명                                         |
| :-------- | :------------------------------------------- |
| ^ + Enter | 해당셀 실행                                  |
| ~ + Enter | 해당셀을 실행하고 커서를 다음 셀로 넘김      |
| @ + Enter | 해당셀을 실행하고 셀을 삽입한 후 커서를 넘김 |
| ^ + M A   | 코드 셀 위에 삽입                            |
| ^ + M B   | 코드 셀 아래 삽입                            |
| ^ + M D   | 셀 지우기                                    |
| ^ + M Y   | 코드 셀로 변경                               |
| ^ + M M   | 마크다운 셀로 변경                           |
| ^ + M Z   | 실행 취소                                    |
