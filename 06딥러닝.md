# 딥러닝

딥러닝은 사람 뇌의 뉴런 구조를 흉내낸 인공신경망(ANN)을 사용합니다.

`개념`

```
AI
 └── 머신러닝
      ├── 분류
      ├── 회귀
      └── 딥러닝
            ├── CNN
            ├── RNN
            └── Transformer
```

```
데이터
 ↓
학습
 ↓
신경망 모델 생성
 ↓
예측
```

`특징`

- 데이터가 많을수록 강력
- 이미지/음성/NLP에서 뛰어남
- GPU 사용이 많음

`딥러닝 예시`

| 분야       | 예시                           |
| :--------- | :----------------------------- |
| 이미지     | MNIST 손글씨 분류, 고양이 판별 |
| 음성       | 음성 인식                      |
| 자연어     | 챗봇, 번역                     |
| 생성형 AI  | ChatGPT, 이미지 생성           |
| 회귀분석   | 영화 리뷰 감성분석             |
| 자연어처리 |                                |

`배워야 할 프레임워크`

| 프레임워크  | 특징 | 비고                                              |
| :---------- | :--- | :------------------------------------------------ |
| TensorFlow  | 구글 | 쉬운 API(입문용). Keras: .TensorFlow용 고수준 API |
| **PyTorch** | 메타 | 실무/연구. 강력추천                               |

`머신러닝과 딥러닝 차이`

| 항목      | 머신러닝       | 딥러닝           |
| --------- | -------------- | ---------------- |
| 특징 추출 | 사람이 직접    | 모델이 자동      |
| 데이터 양 | 적어도 가능    | 많이 필요        |
| 성능      | 일반 문제 강함 | 복잡한 문제 강함 |
| 속도      | 빠름           | 느림             |
| 하드웨어  | CPU 가능       | GPU 권장         |
| 대표 분야 | 표 데이터      | 이미지/음성/NLP  |

## 딥러닝 구조

은닉층이 깊어질수록 "Deep Learning"

```
입력층
  ↓
은닉층(hidden layer)
  ↓
출력층
```

## 시대흐름

```
전통 머신러닝
   ↓
딥러닝 등장
   ↓
CNN (이미지 혁명)
   ↓
RNN (자연어/시계열)
   ↓
Transformer (현재 AI 핵심)
```

## CNN (Convolutional Neural Network)

이미지 처리에 특화된 딥러닝 모델

동작

```
이미지
 ↓
필터(Convolution)
 ↓
특징 추출
 ↓
분류
```

## RNN (Recurrent Neural Network)

- 순서(sequence) 데이터 처리 모델
- 이전 정보를 기억하며 처리
- 예시 : 문장 번역, 음성 인식, 시계열-주가예측
- 문제점 : 문장이 길어질수록 기억 약화 --> 이를 개선한 것이 LSTM, GRU

## Transformer

- 현재 AI 모델의 핵심 구조
- 기존의 RNN의 문제점을 해결 -> 문장 안에서 중요한 단어끼리 관계를 학습
- ChatGPT도 Transformer 기반입니다.
- 활용분야 : 챗봇-ChatGPT, 번역-DeepL, 이미지 생성-Vision Transformer, 음성-Whisper

`Transformer 기반 딥러닝 모델`

| 모델  | 설명        |
| :---- | :---------- |
| GPT   | 생성형 AI   |
| BERT  | 문장 이해   |
| T5    | 번역/요약   |
| ViT   | 이미지 처리 |
| Llama |             |
| T5    |             |

`Transformer 신경망(Neural Network) 내부 구조`

- 뉴런
- 가중치(weight)
- 행렬 연산
- 역전파(backpropagation)
- 학습(train)

## Transformers

```
딥러닝 프레임워크
 ├── TensorFlow
 │      └── Keras
 │
 └── PyTorch

Transformer 모델 라이브러리
 └── Transformers
         ├── GPT
         ├── BERT
         ├── T5
         └── Llama


Keras는 TensorFlow를 쉽게 쓰게 해주는 API,
Transformers는 GPT/BERT 같은 Transformer 모델을 쉽게 사용하는 라이브러리이다.
```

```
pipeline
   ↓
AutoModel
   ↓
PyTorch/TensorFlow

Transformers는 PyTorch/TensorFlow 위에서 동작
TensorFlow와 PyTorch는 딥러닝 엔진이고,
```

`Transformer는 AI 모델 구조이고, Transformers는 Transformer 모델들을 다루는 Python 라이브러리이다. `

- Transformer 모델들을 쉽게 사용하게 해주는 `Hugging Face의 라이브러리`
- 배우기 쉬움: 코드가 간단
- TensorFlow 기반 : 산업 현장에서 많이 사용
- 딥러닝 입문용 좋음 : 학습 곡선이 완만
- GPU 지원 : 대규모 학습 가능

`이 라이브러리를 사용하면`

- GPT 사용
- BERT 사용
- 모델 다운로드
- 토크나이저 사용
- 학습/추론
- 파인튜닝

`이런 사람에게 추천`

- AI 입문자
- 딥러닝 처음 배우는 사람
- 빠르게 모델 실습하고 싶은 사람

`딥러닝 프레임워크와의 관계`

Transformers는 혼자 동작하지 않고 PyTorch, TensorFlow 같은 딥러닝 프레임워크 위에서 동작합니다.

`관계구조`

```
Transformers 라이브러리
 ├── pipeline
 │      └── 매우 쉬운 사용
 │
 └── AutoModel
        └── 모델 직접 제어
```

### pipeline

- AutoModel을 쉽게 감싼(wrapper) API
- 몇 줄로 바로 AI 기능을 사용하는 API

`내부적으로 자동 수행`

- 모델 다운로드
- 토크나이저 다운로드
- 전처리
- 추론
- 후처리

`감정분석 예시`

```python
# pip install transformers
from transformers import pipeline

classifier = pipeline("sentiment-analysis")

result = classifier("I love AI")
print(result)

```

### pipeline 과 AutoModel

| 항목         | pipeline  | AutoModel   |
| ------------ | --------- | ----------- |
| 난이도       | 쉬움      | 어려움      |
| 목적         | 빠른 사용 | 세밀한 제어 |
| 자동화       | 매우 많음 | 거의 없음   |
| 커스터마이징 | 제한적    | 매우 높음   |
| 실무 연구    | 적음      | 많음        |
| Fine-tuning  | 부적합    | 적합        |

pipeline = AutoTokenizer + AutoModel

### tensorflow + Keras

- Keras는 딥러닝 모델을 쉽게 만들 수 있도록 도와주는 Python 라이브러리입니다.
- 신경망(Neural Network)을 간단한 코드로 구현 가능
- 초보자도 배우기 쉬움
- 현재는 주로 Google 의 TensorFlow 와 함께 사용됩니다.

`Keras로 할 수 있는 것`

- 이미지 분류
- 자연어 처리
- 추천 시스템

```python
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

# 1. 신경망 만들기
model = Sequential([
    Dense(64, activation='relu'),
    Dense(1)
])

# 모델 학습
model.fit(x_train, y_train, epochs=10)

# 모델 예측
pred = model.predict(x_test)
```
