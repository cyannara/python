### 상관관계 분석과 상관 계수

#### 상관분석

추천서비스 : 데이터를 분석해서 사용자에게 필요한 정보를 '맞춤형'으로 제공. 가장 많이 사용되는 분석 방법 중 한 가지가 `상관관계 분석`
ex) 키가 크면 발이 크다
교육 수준이 높을수록 자녀의 대학 진학률이 높다.

두 개의 변수에 어떤 관계가 있는지 셩향을 분석하는 것이 상관분석이다. 이를 `산포도` 또는 '산점도' 라는 그래프로 그리면 직관적으로 두 변수 사이의 관계를 파악할 수 있다.

- 양의 상관관계: x의 값이 증가함에 따라 y의 값이 증가하는 경우. 키 vs 발크기
- 음의 상관과계: x의 값이 증가함에 따라 y의 값이 감소하는 경우. 산의 높이 vs 온도

#### 상관계수

- 상관계수(Correlation Coefficient)는 두 변수 간의 관계 강도와 방향
- 점들이 모여 있는 근접도(밀도)를 함께 포함하여 표현.
- 상관계수는 -1~1의 값을 가지고 수치가 0에 가까울수록 두 변수 사이에 상관관계가 없음을 의미. -1에 가까울수록 음의 상관관계가 강하며 1에 가까울수록 음의 상관관계에 가깝다.

| 값          | 의미                    |
| :---------- | :---------------------- |
| 0.8 ~ 1     | 매우 강한 양의 상관관계 |
| 0.6 ~ 0.8   | 강한 양의 상관관계      |
| 0.4 ~ 0.6   | 상관관계가 있다         |
| 0.2 ~ 0.4   | 약한 상관관계           |
| 0           | 관계 없음               |
| -0.6 ~ -0.8 | 강한 음의 상관 관계     |
| -0.8 ~ -1   | 매우 강한 음의 상관관계 |

### pandas 패키지

데이터를 엑셀스프레드시트나 SQL 테이블과 같이 서로 다른 속성을 가지는 컬럼들로 구성된 데이터 테이블 형태(DateFrame)나 시계열 데이터와 같이 연속된 데이터(Series)으로 구성하고, 이를 분석하기 위한 도구로 사용된다.

- Series : 1차원 배열
- DataFrame : 2차원 배열

#### 난수

| 함수         | 분포           | 범위     |
| :----------- | :------------- | :------- |
| randn()      | 정규분포       | 제한없음 |
| rand()       | 균등분포       | 0~1      |
| randint()    | 균등분포(정수) | 지정범위 |
| uniform(a,b) | 균등분포       | a~b      |

균등분포 : Uniform Distribution. 지정범위 안의 값들이 균일한 확률로 생성된다. 구간 전체에서 확률밀도가 일정
uniform : 균일한, 고른

정규분포 : Normal Distribution. 평균을 중심으로 좌우 대칭인 종 모양을 가진 가장 대표적인 연속 확률분포. 자연 및 사회 현상의 데이터 분포 설명. 평균과 표준편차에 의해 형태가 결정

```
확률
 ^
 | █████████████
 | █████████████
 | █████████████
 +----------------> 값
```

| 종류   | 의미                                                                   |
| :----- | :--------------------------------------------------------------------- |
| Scalar | 단일 값. 수학에서 방향의 구별은 없고 하나의 수치만으로 완전히 표시되는 |
| Vector | 여러 값의 집합(1차원 배열). 크기와 방향으로 정해지는 양                |
| Matrix | 2차원 데이터(2차원 배열. 행렬)                                         |

### 벡터

- 방향과 크기를 가진 양. 스칼라는 방향없이 크기만 있는 양. 벡터와 스칼라의 차이는 바로 방향이 있는냐 없느냐입니다.
- 화살표로 표현. 길이가 벡터의 크기(세기). 화살표가 가리키는 방향이 벡터의 방향
- (3,4) 의 벡터의 크기는 피타고라스 정리를 하용해 계산하면 5입니다.
- 실생활 예시 : 스마트폰- 화면에서 손가락을 오른쪽으로 쭉 밀면 그 방향과 세기가 벡터로 표현

#### 딥러닝에서 벡터

- 데이터를 표현하는 기본 단위. 텍스트, 이미지, 음성 등 모든 종류의 데이터가 벡터로 변환
- 컴퓨터는 데이터를 처리하기 위해 수치적 표현을 필요로 합니다. "사과"라는 단어라 고양이 사진을 컴퓨터로 처리하기 위해서는 숫자로 변환. 이 때 숫자하나(스칼라)로는 부족하고 여러 숫자의 조합(벡터)이 필요합니다.
- 딥러닝에서는 매우 높은 차원의 벡터를 사용
  - 워드 임베딩 : 단어는 100~300차원, 이미지 28\*28 픽셀의 흑백 이미지는 784 차원의 벡터로 표현. 각픽셀의 밝기가 하나의 차원 값

### 삼각비, 삼각함수

직각삼각형의 변의 길이 비율을 이용해 각도와 거리를 계산하는 삼각함수. 건축, 항해, 물리학 등 다양한 분야에서 활용

##### 코사인(cos)

- 빗변과 밑변의 비율
- 활용: 거리 측정. 수평 성분 계산, 구면상의 위치파악
- 예시: 항해 시 배의 위치 파악. 건축물의 경사면 길이이와 수평거리 계산. 2D/3D 게임 그래픽에서 물체의 위치 회전

##### 사인(sin)

- 빗변과 높이의 비율
- 활용: 파동(빛,소리, 전파), 주기적인 움직임. 높이 측증
- 예시: 비행기의 이착륙 각도에 따른 상승높이 계산.파도나 교류 전기의 그래프를 그릴 때 사용

##### 탄젠트(tan)

- 밑변과 높이의 비율
- 활용: 기울기 계산, 그림자 길이. 직접 재기 힘든 높이나 거리 측정
- 예시: 산의 높이나 교량의 길이를 측량. 오르막길의 경사도 계산. 게임에서 캐릭터가 바라보는 시야각 계산

##### 요약

- 높이/파동 -> 사인(높이/빗변)
- 수평 거리/위치 -> 코사인(밑변/빗변)
- 기울기/그림자 -> 탄젠트(높이/밑변)
